# Dessiner dans un canvas
La canvas est une balise HTML dont le contenu est détaché du DOM. Il est manipulé par Javascript et possède ses propres paramètres comme la vitesse de rafraîchissement par exemple.
Un canvas est considéré par le navigateur comme une image (faites un clic droit sur un canvas vous verrez, on peut enregistrer l'image actuelle). Il a vocation à recevoir des animations.
## L'exercice
Il s'agira pour vous de :
- créer une balise canvas dans le corps de la page en utilisant Javascript ;
- de charger une image et de l'afficher dans le canvas.
